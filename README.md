# Yellow MVC Framework 
A Simple PHP MVC Framework with Simple blog.

![showcase](https://i.imgur.com/Cd8knMA.png)

## Installation
Clone the repository using command:

    git clone https://github.com/themeyellow/yellow-mvc-framework.git

    composer install
